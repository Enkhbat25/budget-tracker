# Budget Tracker

A single-file HTML application for tracking personal expenses and managing your budget.

## Features (Planned)

- [ ] Add expenses with amount, description, date, and category
- [ ] View all expenses in a list/table
- [ ] Manage spending categories
- [ ] See monthly spending summaries
- [ ] Visualize spending with charts
- [ ] Data persists in browser localStorage
- [ ] Filter and search expenses

## Usage

Open `index.html` in any modern web browser. No server required.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- localStorage for data persistence
